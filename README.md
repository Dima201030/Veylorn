# Veylorn

**Veylorn** — это простой **ASCII roguelike** на C++, где пока что можно **ходить по карте**, исследовать лабиринт и использовать свои карты.  
Каждая карта — это текстовый файл, который можно загрузить в игру.  

---

## Игровая механика

- Игрок может **двигаться по карте**, открывая новые участки.  
- Карты хранятся в текстовых файлах (`map1.txt`, `map2.txt` и т.д.).  
- Можно **добавлять свои карты** в проект, главное — чтобы они были валидными.  
- Игра проверяет карту на корректность перед запуском, чтобы ничего не сломалось.  

> На данном этапе игры это **практический roguelike**, где акцент на исследование карты и возможность подключать свои уровни. В будущем планируется добавить больше механик.  

---

## Запуск

### 1. Сборка и запуск по умолчанию

```bash
clang++ -std=c++17 main.cpp map.cpp lvlparser.cpp -o Veylorn
./Veylorn
```
  •	Без параметров запускается карта по умолчанию.
### 2. Запуск с конкретной картой
```
./Veylorn map1.txt
```

- Можно указать любую карту (map2.txt, map3.txt или свою).
- Важно: без кавычек, просто имя файла.
- Если карта некорректная, игра выдаст ошибку проверки и не запустится.

---

### Особенности
- ASCII-графика: удобно тестировать прямо в терминале.
- Проверка карт на валидность перед запуском.
- Можно добавлять свои карты без изменения кода.
- Roguelike в стиле «explore & walk»: исследуй карту и знакомься с её лабиринтами.

### От автора

Veylorn — это проект для изучения C++ и roguelike-логики.
Сейчас игра минималистична: только передвижение, карты и проверка.

Но самое главное — вы можете создавать свои карты и исследовать их, делая игру немного уникальной каждый раз.

Пробуй создавать новые карты, делись своими лабиринтами, и со временем здесь появится больше механик. Это ваша маленькая roguelike-песочница!
